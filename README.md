# e2e-commerce-automation-framework
End-to-End Test Automation Framework using Selenium, Postman &amp; CI/CD (SDET Portfolio Project)

=> E2E Commerce Automation Framework

-> Overview
This project is a scalable End-to-End Test Automation Framework built using:
* Selenium WebDriver (Java)
* TestNG
* Postman + Newman
* Maven
* Allure Reporting
* GitHub Actions (CI/CD)

-> Project Objective
To demonstrate real-world automation practices covering:
* UI test automation
* API test automation
* CI/CD integration
* Reporting & logging
* Data-driven testing
* Retry mechanism for flaky tests

-> Architecture
* Page Object Model (POM)
* Utility classes
* Config-driven execution
* Modular test structure

-> Test Coverage

> UI Automation (SauceDemo)
* Login
* Product Search
* Add to Cart
* Checkout

> API Automation (ReqRes)
* Login API
* Get Users
* Create User
* Token validation
